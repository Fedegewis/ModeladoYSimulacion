2. ¿Cómo encuentro las raíces algebraicamente?
Para f(x) = (x + 2)(x + 1)(x - 1)³(x - 2), uso la propiedad del producto cero:
Un producto es cero ⟺ Al menos uno de los factores es cero
Por tanto:

(x + 2) = 0  →  x = -2
(x + 1) = 0  →  x = -1
(x - 1)³ = 0  →  x = 1
(x - 2) = 0  →  x = 2

4. ¿Cómo sé si cruza o toca el eje?
Regla fundamental:

Multiplicidad IMPAR (1, 3, 5...): La función CRUZA el eje X
Multiplicidad PAR (2, 4, 6...): La función TOCA el eje X pero no cruza

En nuestro caso:

x = -2, -1, 2: multiplicidad 1 (impar) → CRUZAN el eje
x = 1: multiplicidad 3 (impar) → CRUZA el eje (pero con forma más suave)

5. ¿Cómo verifico convergencia usando el Teorema del Valor Intermedio?
Teorema: Si f es continua en [a,b] y f(a) × f(b) < 0, entonces ∃c ∈ (a,b) tal que f(c) = 0
python# Ejemplo: intervalo [-3, 2.5]
